# RHFP Project

## 📌 Description
This is a Product Review System where users can add reviews and admin approves them.

## 🚀 Tech Stack
- React (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

## 🔑 Features
- User Authentication (JWT)
- Add Product Reviews
- Admin Approval System
- Rating Calculation

## ⚙️ Setup Instructions

### Backend
- Set environment variables:
  DB_USERNAME, DB_PASSWORD
- Run:
  mvn spring-boot:run

### Frontend
- Run:
  npm install
  npm run dev
